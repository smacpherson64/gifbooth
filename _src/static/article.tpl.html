GIFBooth

GIFBooth is an experiment using various Javascript API's to allow users to use live video to create animated GIFs. Selfies with some magic.

Tools

Gulp ETA
An open source frontend build script developed by 40Digits
GIF.js
Javascript library for creating animated GIFs made by Johan Nordberg
html5sortable
jQuery pluggin for sorting lists and grids.

Walking through error reporting: Google Analytics Exceptions , also Google Analytics Report on Stack Overflow.
